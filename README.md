# Sales Data Analysis (Pandas)

This project performs basic data analysis on a sales dataset using Python and pandas.

## Features
- Load and inspect sales data
- Calculate total revenue
- Analyze revenue by category
- Find top-selling products
- Export results to CSV files

## Project Structure

pandas-data-analysis-project/
- data/
  - sales.csv
- output/
  - revenue_by_category.csv
  - top_products.csv
- analysis.py
- requirements.txt
- README.md

## Installation

```bash
pip install -r requirements.txt
```

## How to Run

```bash
python analysis.py
```

## Example Results

- Total Revenue: 1925
- Electronics Revenue: 875
- Furniture Revenue: 1050

- pandas
